# Educational Attainment and Self-Rated Health (NHIS 2022)

This project analyzes the relationship between educational attainment and self-rated health among U.S. adults using NHIS 2022 data.

## Methods
- Ordinal logistic regression (SAS)
- Outcome: self-rated health
- Predictor: education
- Covariates: age, sex, work income
